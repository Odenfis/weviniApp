## 📌 Descripción del Proyecto

weviniApp es una aplicación web moderna diseñada para ofrecer una experiencia eficiente, escalable y centrada en el usuario. Este proyecto integra tecnologías actuales tanto en frontend como backend, siguiendo buenas prácticas de desarrollo y despliegue.

---

## 🧩 Arquitectura

El proyecto está dividido en dos partes principales:

- 🔹 Frontend: Interfaz de usuario moderna
- 🔹 Backend: API y lógica del servidor
- 🔹 Docker: Contenerización completa del sistema

---

## 🛠️ Tecnologías utilizadas

### Frontend
- HTML5, CSS3, JavaScript
- Vite
- (Posible uso de frameworks modernos)

### Backend
- Node.js
- Express

### DevOps
- Docker
- Docker Compose

## ⚙️ Instalación y ejecución

### 🔧 Requisitos previos
- Node.js
- Docker (recomendado)

---

### ▶️ Ejecución con Docker (recomendado)

bash docker-compose up --build 

---

### ▶️ Ejecución manual

#### 1. Instalar dependencias
bash npm install 

#### 2. Ejecutar frontend
bash npm run dev 

#### 3. Ejecutar backend
bash cd server npm install npm run start 

---

## 🌱 Variables de entorno

Copia el archivo .env.example y crea tu propio .env:

bash cp .env.example .env 

Configura tus variables según el entorno.
